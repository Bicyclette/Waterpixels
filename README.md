# Waterpixels
University project : Image segmentation using the watershed algorithm.<br/>
C++ implementation based on the Waterpixels research paper readable at the link below :<br/>
[https://hal.archives-ouvertes.fr/hal-01212760/document](https://hal.archives-ouvertes.fr/hal-01212760/document)<br/>

# Dependencies
+ OpenMP
+ Qt5
+ OpenCL

# How to build(GNU/Linux systems)
1) in the root directory run the python script *./generate_makefile.py*
2) go to build directory and compile with *make*
3) run the software : ./Waterpixels
